# Accordion

Accordions allow you to reduce vertical space by creating collapsible sections in your content while also organizing and grouping information.

<CH.Code>

```jsx main.tsx
import { Accordion, AccordionPanel } from "@fluid-design/fluid-ui";
import { data } from "./data";

<Accordion>
  {data.map((item, index) => (
    <AccordionPanel key={index} header={item.title} isOpen={item.isOpen}>
      {item.details}
    </AccordionPanel>
  ))}
</Accordion>;
```

## With Icons

<CH.Section>

You can use [`headerIcon`](focus://main.tsx#7) to add an icon to the header.
The icon needs to be a function, you can import them from libraries such as [react-icons](https://react-icons.github.io/) or [heroicons](https://heroicons.com/).

<CH.Code>

```jsx main.tsx
<Accordion>
  {dataWithIcons.map((item, index) => (
    <AccordionPanel
      key={index}
      header={item.title}
      isOpen={item.isOpen}
      // mark
      headerIcon={item.Icon}
    >
      {item.details}
    </AccordionPanel>
  ))}
</Accordion>
```
