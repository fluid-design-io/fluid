# 展示页面

## 基本用法

这是一个展示页面。这个页面包含了如下功能：

- 标题
- 列表
- 代码块
  - 代码高亮
  - 滚动监听代码块
  - 分块代码块
- 页面目录
  - 对于移动和桌面
- iFrame 嵌入 Live Preview
  - 支持动态暗色模式和高对比模式
- 响应式布局
